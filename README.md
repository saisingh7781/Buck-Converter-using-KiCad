# Buck-Converter-using-KiCad
A Buck Converter PCB designed using KiCad, including the schematic, PCB layout, footprints, and 3D board visualization.

📖 Overview

This project demonstrates the design of a Buck Converter (DC-DC Step-Down Converter) using KiCad. The project covers the complete PCB design workflow, from schematic capture to PCB layout and 3D visualization.

🎯 Objectives

Design a Buck Converter schematic.
Assign component footprints.
Create a PCB layout.
Perform PCB routing.
Verify the design using DRC/ERC.
Generate a 3D PCB model.

🛠️ Software Used

KiCad
🔧 Components Used
IRLZ44N N-Channel MOSFET
100 µH Inductor
1N5819 Schottky Diode
220 µF Electrolytic Capacitor
10 Ω Resistor
2-Pin Power Connector
